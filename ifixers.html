iFixers - iOS App

iOS Mobile Design, Visual & Interaction Design, UX

This project was requested by a client who owns an iPhone repair shop.
We started sketching out the basic wireframes as we brainstormed.

I considered the screen sizes we're designing for, in order to create a template that's close to the final size.

I've decided that I'll use the size of iPhone 5 and 5s as a baseline.

Then I took a first pass at the main screens.

After a few waves of feedback, I've defined the user flow.
